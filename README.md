# 🌍 World Crypto Data Analysis Project (MS Excel)

## 📌 Overview
This project focuses on analyzing global cryptocurrency market data using **Microsoft Excel**. It involves data collection, cleaning, transformation, and visualization to extract meaningful insights about crypto market trends.

The project demonstrates how Excel can be used as a powerful data analytics tool for real-world financial datasets.

---

## 📂 Project Structure

```
├── crypto_data.xlsx        # Raw dataset
├── Book1.xlsx             # Processed data & dashboard
├── new.ipynb              # Data preprocessing
├── practice.ipynb         # Practice work
├── scrap_sample.ipynb     # Web scraping sample
├── Web_Scraping.zip       # Scraping scripts
└── README.md              # Project documentation
```

---

## 🎯 Objectives

- Analyze top cryptocurrencies based on:
  - Market Capitalization  
  - Trading Volume (24h)  
  - Price trends  
- Identify patterns in crypto market behavior  
- Build an interactive Excel dashboard  
- Apply data filtering and conditional logic  

---

## 🛠️ Tools & Technologies

- **Microsoft Excel**

  - Conditional Formatting  
  - VBA (Automation)

  - Data Collection (Web Scraping)

## 📊 Features

### ✅ Data Analysis
- Sorting & filtering of crypto coins  
- Identification of top-performing cryptocurrencies  
- Volume and price comparison  

### 📈 Dashboard
- Interactive charts:
  - Top 10 cryptocurrencies  
  - Volume trends  

### ⏱️ Automation (VBA)
- Dashboard visible only during working hours (9 AM – 5 PM)  
- Automatic refresh functionality
